# 🎯 QuizMaster - Interactive Quiz Website

QuizMaster is a modern and responsive quiz website built using **HTML**, **CSS**, and **JavaScript**.

This project includes:
- Live Quiz System
- Real-Time Timer
- Progress Tracker
- Dashboard
- Leaderboard
- Responsive Design

---

# 🚀 Features

✅ Modern Responsive Design  
✅ Sticky Navigation Bar  
✅ Smooth Scrolling Navigation  
✅ Interactive Quiz System  
✅ Real-Time Countdown Timer  
✅ Live Progress Tracker  
✅ Attempted & Remaining Questions Counter  
✅ Score Tracking  
✅ Quiz Categories Section  
✅ Dashboard Section  
✅ Leaderboard UI  
✅ Footer with Social Icons  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
